# Choredom-API
Choredom-API is the back end API for the choredom educational application.

## Prequisites
NodeJS  
NPM (Node Package Manager)  
Sequelize-CLI

### Install Dependencies
$ cd /choredom-api/  
$ npm install  
$ sudo npm install -g sequelize-cli

## Running the Development Server
$ cd /choredom-api/  
$ npm run start:dev
